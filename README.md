# 🎲 Teach-M — Language Teacher's Toolbox

Nowoczesna, interaktywna baza **zabaw, gier i metod aktywizujących** na lekcje języków obcych.
Jeden plik HTML, działa offline w przeglądarce — dane zapisują się lokalnie.

## ✨ Funkcje

- 🔍 **Wyszukiwarka** po nazwie, opisie, umiejętnościach i krokach
- 🎛️ **Filtry**: kategoria, poziom (A1–C2), wiek, umiejętność + sortowanie
- ⭐ **Ulubione** — szybki dostęp do sprawdzonych zabaw
- 📋 **Plan lekcji** — koszyk aktywności z łącznym czasem i **wydrukiem/PDF**
- ➕ **Dodawanie własnych zabaw** przez formularz (bez edycji kodu)
- 📄 **Duplikowanie** istniejącej zabawy jako szablonu
- 🎁 **Eksport / import** — pełna kopia zapasowa oraz dzielenie się własnymi zabawami
- 🎰 **Losowanie** zabawy i 🌙 **tryb ciemny**

## 🚀 Użycie

Otwórz `index.html` w przeglądarce (dwuklik) — to wszystko.
Wersja online: **https://dandeliant.github.io/Teach-M/**

## 🧩 Dodawanie zabaw

Dwa sposoby:
1. Przycisk **„➕ Dodaj zabawę"** na stronie (zapis w przeglądarce).
2. W kodzie — tablica `SEED` na dole pliku `index.html` (dobre do większych partii).

## 💾 Dane

Ulubione, plan lekcji i własne zabawy przechowywane są w `localStorage` przeglądarki.
Rób kopie przez przycisk **„💾 Kopia"**, aby przenieść je na inny komputer.
